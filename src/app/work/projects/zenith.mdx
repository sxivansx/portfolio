## Overview

Zenith is a 36-hour national-level tech extravaganza, and this project showcases the complete visual and digital branding—from web experience to print assets—designed to amplify the event’s futuristic, space-themed identity.

## Key Features

- **Futuristic Visual Identity**: Designed a cohesive brand language inspired by the idea of "Zenith" as the highest peak, using space motifs, blue gradients, and bold modern typefaces.
- **Cross-Media Consistency**: Applied the visual system across digital and physical formats, including landing page UI, social banners, ID cards, brochures, and event badges.
- **User-Focused Landing Page**: Built an immersive hero section with countdown timers, motion graphics, and clear CTAs for participation and registration.

## Tools Used

- **Figma**: For wireframes, UI layouts, and design system management.
- **Illustrator & Photoshop**: For graphics, visual exploration, and print-ready assets.

## Challenges and Learnings

The main challenge was building a high-impact identity system that felt both futuristic and accessible, balancing vibrant aesthetics with information clarity across diverse formats.

## Outcome

The branding elevated Zenith's online presence and participant engagement, creating a visually memorable event experience praised by attendees and organizers alike.
